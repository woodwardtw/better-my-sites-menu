== Description ==


Pretty much removes any site from the my sites list where the user doesn't have at least two roles thus preventing the contributor designation junking everything up.

== Installation ==



Using this plugin is very simple. All you have to do is:


1. Upload the folder to the `/wp-content/plugins/` directory

1. Network Activate the plugin through the 'Plugins' menu in WordPress


Thats it!
